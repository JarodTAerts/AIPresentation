concept

Skills — reusable instructions, not new model weights

A Skill is a folder of instructions and supporting files that an agent can load when a task matches. It adds procedural knowledge without retraining the model.

1
Advertise
The agent sees a short name and description: “Use this for incident reports.”
2
Load on demand
When relevant, it reads the full instructions, templates, examples, or scripts.
3
Follow the playbook
The same general model now follows your team’s process and output format.
MCP gives reach.
Connect to live tools and data: GitHub, Slack, files, databases.
Skills give procedure.
Explain how to perform a recurring task well and consistently.
Skills can still contain bad instructions or unsafe scripts. Treat them like code: review the source, pin trusted versions, and keep permissions narrow.

Source: agentskills.io open specification. GitHub Copilot “Skillsets” are a separate 2024 integration feature, not this file format.
