why code first

Code is just structured language with a brutally fast grader.

Code is text…

Same input format as English. The model already knows how to predict the next token. There are ~1 trillion lines of public code on GitHub to learn from.

…with an grader

Did the code compile? Did the tests pass? Instant, automatic feedback. Essays don't have that. This is why models trained on code got dramatically smarter at everything — reasoning, math, even writing.
