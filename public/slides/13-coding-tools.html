the tools

What "AI coding tools" actually are

Three generations, all live in market today.

Gen 1
Autocomplete
GitHub Copilot, Cursor Tab. Suggest the next lines as you type inside the editor.
Gen 2
Chat & edit
Copilot Chat, Cursor. Read files, answer questions, and propose edits for you to review.
Gen 3
Agents
Claude Code, Copilot CLI, Devin, Aider. Edit files, run commands, inspect results, and iterate.
All three generations use the same core idea: gather context → ask a model → apply a suggestion or tool call → inspect the result. The next slide shows that loop in motion.
