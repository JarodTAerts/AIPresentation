MCP servers & Skills — giving models hands

The model is the brain. MCP servers are the hands and senses. Open standard from Anthropic (Nov 2024), now adopted by OpenAI, Google, GitHub, and ~5,000+ public servers.

Model Context Protocol — one socket, many tools.

MCP server

A tiny program that exposes tools (do something), resources (read something), and prompts to any AI client. Same wire format whether you're hitting Slack or your local filesystem.

Skill

A pre-packaged instruction set the model can load on demand — "how to write good unit tests," "how to follow this team's code style," "how to file an incident report." Think of it as muscle memory the model can swap in.

Together: the model brings reasoning. MCP brings reach. Skills bring specialization. That's how a generic chatbot becomes "my company's AI."

Spec: modelcontextprotocol.io · Skills: Anthropic Agent Skills (2025), GitHub Copilot Skillsets.
