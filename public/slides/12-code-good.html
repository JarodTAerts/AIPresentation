why code first

Code is just structured language with a brutally fast grader.

Code is text…

Same input format as English. The model already knows how to predict the next token. There are 500M+ public repositories on GitHub to learn from (Octoverse 2024).

…with an grader

Did the code compile? Did the tests pass? Instant, automatic feedback. Essays don't have that. This is why models trained on code got dramatically smarter at everything — reasoning, math, even writing.

SWE-bench Verified — % of real GitHub issues solved autonomously

Source: swebench.com/verified leaderboard, snapshots Mar 2024 → Feb 2026.

In ~24 months, frontier models went from ~13% on the original SWE-bench (Mar 2024) to 65–75% on the newer SWE-bench Verified (introduced Aug 2024). No other knowledge-work benchmark has moved that fast.
