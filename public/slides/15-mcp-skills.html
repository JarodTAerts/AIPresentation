SECTION IV · THE NEW PLUMBING

MCP — a shared protocol for tools and data

The model supplies reasoning; external software supplies access. Anthropic introduced MCP in Nov 2024, and it is now supported across major model and developer platforms. Public registries list thousands of servers — counts vary by registry and verification method.

Model Context Protocol — one socket, many tools.

MCP server

A program that exposes tools (do something), resources (read something), and prompts through a shared protocol. Authentication and least privilege still matter: an MCP server is software you must trust.

What it standardizes

The client discovers capabilities, sends structured requests, and receives structured results. The same protocol can sit in front of GitHub, a database, or a local filesystem.

Standardized does not mean safe by default. Tool descriptions and returned data can be malicious. Use trusted servers, explicit consent, narrow permissions, and audit logs.
